### 📌 About This Repository

This repository documents everything I build, study, and experiment with as I progress through the ML Zoomcamp curriculum. It includes notebooks, homework solutions, projects, and personal notes organized by module.

The course is taught by **Alexey Grigorev** and the DataTalks.Club team, and covers machine learning end-to-end - not just model building, but the full engineering workflow required to take models to production.

---

### 🎯 What the Course Proposes

The ML Zoomcamp focuses on **ML Engineering** - the practical side of machine learning that bridges the gap between a trained model and a real, deployed application. The course teaches you to:

- Understand and implement the most important ML algorithms from scratch and with libraries
- Evaluate models properly and avoid common pitfalls
- Package, serve, and deploy models as web services
- Scale deployments using cloud infrastructure and orchestration tools

All materials are open-source and available on YouTube. Cohort students also submit projects that are peer-reviewed and can earn a certificate.

### 📚 Curriculum Overview

#### Module 1 - Introduction to Machine Learning
Fundamentals of what ML is, when to use it, and how to frame problems using the **CRISP-DM** methodology (Cross-Industry Standard Process for Data Mining).

**Topics:** ML vs rule-based systems · Supervised learning · CRISP-DM · Model selection · Environment setup (Python, NumPy, Pandas)

---

#### Module 2 - Machine Learning for Regression
Building a **car price prediction** model as a practical introduction to supervised learning.

**Topics:** Linear regression from scratch · EDA · Feature engineering · Regularization (Ridge) · Train/validation/test split · Scikit-learn

---

#### Module 3 - Machine Learning for Classification
Predicting **customer churn** using logistic regression.

**Topics:** Logistic regression · Sigmoid function · One-hot encoding · Feature importance · Model interpretation · Scikit-learn pipelines

---

#### Module 4 - Evaluation Metrics for Classification
Learning how to **properly measure** model performance and handle real-world data challenges.

**Topics:** Confusion matrix · Accuracy · Precision & Recall · F1-score · ROC curve · AUC · Cross-validation · Class imbalance

---

#### Module 5 - Deploying Machine Learning Models
Turning a trained model into a **production-ready web service**.

**Topics:** Model serialization with Pickle · FastAPI · REST APIs · Docker containerization · Cloud deployment

---

#### Module 6 - Decision Trees & Ensemble Learning
Exploring more powerful, non-linear algorithms and ensemble methods.

**Topics:** Decision trees · Random Forest · Gradient Boosting · XGBoost · Hyperparameter tuning · Feature importance

---

#### ⚡ Midterm Project
An independent end-to-end ML project applying everything from Modules 1–6: data collection, EDA, model training, evaluation, and deployment as a web service.

---

#### Module 8 - Neural Networks & Deep Learning
Introduction to deep learning for image and structured data tasks.

**Topics:** Neural network fundamentals · Activation functions · Backpropagation · Convolutional Neural Networks (CNNs) · Transfer learning · TensorFlow & Keras · PyTorch

---

#### Module 9 - Serverless Deep Learning
Deploying ML models without managing servers using **AWS Lambda**.

**Topics:** Serverless architecture · AWS Lambda · API Gateway · Deploying Scikit-learn, TensorFlow, and PyTorch models serverlessly

---

#### Module 10 - Kubernetes & TensorFlow Serving
Scaling model serving for production using container orchestration.

**Topics:** Kubernetes basics · TensorFlow Serving · Horizontal scaling · Load balancing · kubectl · Kind (local K8s)

---

#### 🏁 Capstone Projects (×2)
Two final projects showcasing the complete ML engineering pipeline - from raw data to a deployed, scalable application.
